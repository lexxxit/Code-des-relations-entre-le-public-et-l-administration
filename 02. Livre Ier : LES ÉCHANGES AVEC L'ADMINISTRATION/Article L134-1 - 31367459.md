# Article L134-1

Sans préjudice de dispositions particulières figurant dans d'autres textes, le présent chapitre régit les enquêtes publiques
qui doivent être organisées par l'administration et qui ne relèvent ni du code de l'expropriation pour cause d'utilité
publique ni du code de l'environnement.
